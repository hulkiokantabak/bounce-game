Enable to watch an AI agent play the game automatically. It places surfaces to guide the ball toward the ring.

Developer API

Control the game programmatically via the browser console:

window.BounceAgent.getState()
window.BounceAgent.tap(x, y)
window.BounceAgent.startRun()
window.BounceAgent.getSummary()

Connect an AI provider:
window.BOUNCE_AI = { onGameState(s) { ... } }
